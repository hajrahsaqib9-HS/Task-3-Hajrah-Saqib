# Mood Bloom
Mood Bloom is an interactive mood tracking web application designed to help users monitor their emotions and gain insights through an analytical dashboard. The application provides authentication features and a user-friendly interface for a seamless experience.

## Features
- User Registration (Sign Up)
- User Login System
- Interactive Dashboard
- Mood Tracking and Analysis
- Analytical Reports Section
- Responsive Design
- Clean and Modern User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
- index.html
- login.html
- signup.html
- dashboard.html
- analytical.html
- style.css
- script.js

## Purpose
The purpose of Mood Bloom is to provide users with a platform where they can track their moods, analyze emotional patterns, and improve self-awareness through visual insights.
